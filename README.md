### 2. Set up product catalog

#### Clone locally

1. Clone the repository you created earlier.

   ```sh
   git clone https://github.com/PATH_TO_YOUR_REPO
   ```

2. Install dependencies using npm, Yarn, or pnpm.

   Install using npm:

   ```sh
   npm install
   ```

   Install using Yarn:

   ```sh
   yarn install
   ```

   Install using pnpm:

   ```sh
   pnpm install
   ```
